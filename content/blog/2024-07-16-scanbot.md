**Scanbot** is a scanning tunneling microscopy automation workflow built around the **nanonisTCP** Python module for programmatic control of Nanonis. It supports scripted **grid scanning** and batch acquisition for long-duration experiments, reducing operator overhead and improving reproducibility.

**DOI:** [10.21105/joss.06028](https://doi.org/10.21105/joss.06028)
